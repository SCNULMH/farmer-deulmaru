# 발표자료 구성안

## 1. 표지

- 제목: 들마루 v2
- 부제: 청년농 맞춤 공공데이터 기반 농업 의사결정 대시보드
- 팀명/팀원명

## 2. 문제 정의

청년농과 귀농 초기 사용자는 지원사업, 재배 일정, 병해충 정보를 각각 다른 사이트에서 찾아야 합니다. 정보는 존재하지만 개인 상황에 맞게 연결되어 있지 않아 실제 의사결정까지 이어지기 어렵습니다.

## 3. 해결 아이디어

들마루 v2는 사용자의 지역과 재배 작물을 기준으로 지원사업, 작물 일정, 병해충 정보, AI 진단 이력을 한 화면에서 제공합니다.

## 4. 공공데이터 활용

| 데이터 | 출처 | 활용 |
| --- | --- | --- |
| 청년농 지원사업 | 농림축산식품 공공데이터 | 지원사업 추천, 관심 등록 |
| 작물 재배 일정 | 농사로 | 이번 주 작업 요약 |
| 병해충 정보 | NCPMS | 진단 결과와 예찰 정보 연결 |
| 지역 지원 정보 | 지자체/공공데이터 | 지역 기반 확장 |

## 5. 서비스 흐름

1. 로그인
2. 개인화 대시보드 확인
3. 지원사업 관심 등록
4. 작물 재배 일정 확인
5. 이미지 업로드 및 AI 진단
6. 진단 이력 저장

## 6. 주요 화면

- 로그인 화면
- 대시보드 Hero
- 지원사업 추천 카드
- 재배 일정 타임라인
- AI 진단 폼
- 진단 이력 카드

## 7. 기술 구조

- Frontend: Jinja2, HTML, CSS, JavaScript
- Backend: FastAPI
- DB: SQLite 로컬 개발, Firebase Firestore 제출/배포
- Hosting: Firebase Hosting + Cloud Run
- AI: PyTorch 모델 어댑터 연결 예정
- Secrets: `.env`, Cloud Run Environment Variables

## 8. 현재 구현 상태

- FastAPI 프로젝트 구조 생성
- 로그인/로그아웃
- SQLite 사용자 DB 및 Firebase 전환 구조
- 관심 지원사업 저장
- AI 진단 결과 저장
- 진단 이력 조회
- Firebase Hosting/Cloud Run 배포 구성

## 9. 차별성

기존 농업 정보 서비스는 사용자가 직접 찾아야 하는 조회형 서비스입니다. 들마루 v2는 청년농의 상황에 맞게 정보를 묶어 “오늘 무엇을 해야 하는지”를 알려주는 의사결정형 서비스입니다.

## 10. 사업화 및 확장

- 청년농 정착 지원 서비스
- 지자체 지원사업 안내 플랫폼
- 농업 교육기관 실습 관리 도구
- 병해충 진단 이력 기반 상담 연계

## 11. 기대 효과

- 정보 탐색 시간 단축
- 지원사업 신청 기회 확대
- 병해충 대응 속도 향상
- 공공데이터 활용 가치 증대

## 12. 마무리 문장

들마루 v2는 흩어진 농업 공공데이터를 청년농의 실제 의사결정 흐름으로 연결해, 농업 정착의 첫 단계를 더 쉽게 만드는 서비스입니다.
